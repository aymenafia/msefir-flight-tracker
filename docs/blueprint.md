# **App Name**: Msefir Flight Tracker

## Core Features:

- Flight Search: Search for flights by flight number, automatically splitting airline code and number.
- Flight Status Display: Display flight details including flight number, airline, airports, scheduled and estimated times, and status.
- Favorites Management: Allow users to star flights and store them in localStorage for easy access.
- Manual Refresh: Implement a refresh button with throttling to prevent excessive API calls.
- Error Handling: Provide friendly error messages for common issues like flight not found or service unavailable.
- Flight Status API Integration: Retrieve flight status from Aviationstack API and normalize the response.
- Temporary Community Rooms: Allow passengers to create temporary community rooms where they can post updates.

## Style Guidelines:

- Primary color: Deep Blue (#0F2A44) for trust and aviation.
- Background color: Off-white (#F7F9FC), providing a calm and clean backdrop.
- Accent color: Sky Blue (#5DA9E9) for interactive elements and highlights.
- Body and headline font: 'Inter', a modern sans-serif for clear readability.
- Simple outline icons (plane, clock, alert, refresh, star) for intuitive navigation.
- Rounded cards and soft shadows to create a modern and approachable interface.
- Subtle animations for loading states and transitions to enhance the user experience.
